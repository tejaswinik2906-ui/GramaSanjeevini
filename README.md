# Grama-Sanjeevini

AI-powered Rural Pharmacy Network Android Application.

## 📱 Features
- Medicine Search
- Nearby Pharmacy Detection
- Firebase Authentication
- Emergency Medicine Alerts
- Multi-language Support
- Google Maps Navigation
- Inventory Management
- Rural Healthcare Support

## 🛠 Technologies Used
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- Google Maps API
- Android Studio

## 🎯 Purpose
This project helps rural people find nearby medical stores and medicine availability quickly.
